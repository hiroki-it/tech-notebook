## 01. Redshift

### Redshiftとは

データウェアハウスとして働く。

データベースよりも柔軟性の高い保管形式で処理済みのデータを管理できる。

<br>

### 高速処理

Apache Sparkを使用して、大きなサイズのデータを高速に並列処理できる。
