## 01. viteコマンド

### build

本番環境用にJavaScriptファイルをビルドする。

```bash
$ vite build
```

> - https://vite.dev/guide/cli#vite-build

<br>

### dev

開発環境用にJavaScriptファイルをビルドする。

```bash
$ vite dev

# または
$ vite
```

> - https://vite.dev/guide/cli#vite
