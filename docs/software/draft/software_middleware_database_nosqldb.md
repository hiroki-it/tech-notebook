### アーキテクチャ

### NoSQL

#### ▼ NoSQLとは

NoSQLは、データ同士が関係を持たないデータ格納形式である。

RDBとは異なり、データをメインメモリに保管する。

#### ▼ NoSQLの種類

代表的なものとして、以下がある。

- Memcached
- Redis
- AWS DynamoDB
- MongoDB
- Cassandra
