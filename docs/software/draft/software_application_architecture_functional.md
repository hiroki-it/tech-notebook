## 01. 副作用

同じ入力であっても出力が異なっていたり、外部の状態に依存するロジックのこと。

- ファイルやデータベースへの書き込み
- HTTPリクエストや外部APIとの通信
- DOMの操作や画面描画
- グローバル変数や外部スコープの変更
- ログ出力や例外のスロー

関数型では、副作用のないロジックを持つ関数を定義する必要がある。
