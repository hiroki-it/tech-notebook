## 01. データレイク

### データレイクとは

データ分析に使用する生データを保管する。

様々な形式のデータを保管できる。

> - https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-a-data-lake

<br>

### 管理データの種類

- ビッグデータ
- IoT
- SNS
- ストリーミングデータ

> - https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-a-data-lake

<br>

## 02. データウェアハウス

### データウェアハウスとは

加工済みデータを保管する。

特定の形式のデータのみを保管できる。

> - https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-a-data-lake

<br>

### 管理データの種類

- アプリケーションデータ
- ビジネスデータ
- トランザクションデータ
- バッチ出力データ

> - https://azure.microsoft.com/en-us/resources/cloud-computing-dictionary/what-is-a-data-lake
