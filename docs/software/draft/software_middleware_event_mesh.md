## 03. イベントメッシュ

### イベントメッシュとは

マイクロサービス間の通信方式でイベント駆動方式を採用した場合に使用するメッシュ。

サービスメッシュツールを使用してもイベント駆動方式を使用できるが、イベントメッシュをツールを使用する方が良い。
