## 03. イベントメッシュ

### イベントメッシュとは

マイクロサービス間の通信方式でパブリッシュ/サブスクライブパターンを採用した場合に使用するメッシュ。

サービスメッシュツールを使用してもパブリッシュ/サブスクライブパターンを使用できるが、イベントメッシュをツールを使用する方が良い。
