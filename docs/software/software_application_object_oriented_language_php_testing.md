## 01. コードベースのテスト

### テスト手順

1. ソースコードを整形する．
2. ソースコードの静的解析を行う．
3. ユニットテストと機能テストを行う．

<br>

### 整形ツール

PhpStorm，PHP-CS-Fixer

<br>

### 静的解析ツール

PhpStorm，PHPStan，Larastan

<br>

### ユニットテストツール，機能テストツール

PHPUnit

<br>

## 02. テスト仕様書ベースのテスト
