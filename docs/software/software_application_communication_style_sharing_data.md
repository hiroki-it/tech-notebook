## 01. データ共有方式

### データベースを介した共有

分析サービスが処理結果データを非同期でストレージに保存し、ほかの業務サービスがこれを使用する。

<br>

### ファイルストレージを介した共有

バッチサービスが処理結果ファイルを非同期でストレージに保存し、ほかの業務サービスがこれを使用する。
