## 03. サブスクライプ

### プル型

プル型のサブスクライブの場合、サブスクライブによる購読予約を Kafka に実行して予約し、そのうえで Kafka にポーリング (Kafka Protocol) を実行する。

注意点として、RabbitMQ のプル型では、サブスクライバーは RabbitMQ にポーリング (HTTP、AMQP など) を実行し、メッセージの取得を待機する（サブスクライブによる購読予約はない）。
