## 03. サブスクライプ

### プル型

プル型のサブスクライブの場合、サブスクライブによる購読予約をKafkaに実行して予約し、その上でKafkaにポーリングを実行する。

注意点として、RabbitMQのプル型では、サブスクライバーはRabbitMQにポーリングを実行し、メッセージの取得を待機する（サブスクライブによる購読予約はない）。
