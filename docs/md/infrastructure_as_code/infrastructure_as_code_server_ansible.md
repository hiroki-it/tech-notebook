## Ansibleとは

### ```playbook.yml```ファイル

#### ・```playbook.yml```ファイルとは

ソフトウェアのインストールタスクの手順を設定する。

<br>

### ```inventory/*```ファイル

#### ・```inventory/*```ファイルとは

反映先のサーバーの情報を設定する。

<br>

### ```group_vars/*```ファイル

#### ・```group_vars/*```ファイルとは

複数の仮想サーバーを構築する。

<br>

### ```host_vars/*```ファイル

#### ・```host_vars/*```ファイルとは

単一の仮想サーバーを構築する。
