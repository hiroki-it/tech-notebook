## 03. 設定ファイル

### ```playbook.yml```ファイル

ソフトウェアのインストールタスクの手順を設定する．

<br>

### ```inventory/*```ファイル

反映先のサーバーの情報を設定する．

<br>

### ```group_vars/*```ファイル

複数の仮想サーバーを構築する．

<br>

### ```host_vars/*```ファイル

単一の仮想サーバーを構築する．
