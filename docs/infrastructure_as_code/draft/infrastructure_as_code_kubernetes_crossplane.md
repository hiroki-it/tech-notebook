## 01. Crossplaneとは

既存のKubernetesリソースのAPIを拡張する。

また、Kubernetesの外部プロバイダー (例：AWS、Google Cloud、Azure、など) のリソースをKubernetesリソースとして管理できるようにする。

> - https://docs.crossplane.io/latest/concepts/
